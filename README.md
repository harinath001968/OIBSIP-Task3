# OIBSIP Task 3 - Temperature Converter Website

## Project Overview
This project was developed as Task 3 of the Oasis Infobyte Web Development and Designing Internship Program.

The Temperature Converter Website allows users to convert temperatures between Celsius, Fahrenheit, and Kelvin using a simple and responsive user interface.

## Features
- Convert Celsius to Fahrenheit and Kelvin
- Convert Fahrenheit to Celsius and Kelvin
- Convert Kelvin to Celsius and Fahrenheit
- Input validation
- Reset button
- Responsive Design
- Modern User Interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
